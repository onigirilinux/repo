OnigiriLinux Repository

About

This is the official package repository for OnigiriLinux. It contains custom packages and plugins for the OnigiriLinux distribution.

Usage

To use this repository, add the following to your /etc/pacman.conf:

[onigirilinux]
SigLevel = Optional TrustAll
Server = https://onigirilinux.github.io/repo/$arch

Available Packages

  • onigiri-updater - System update manager
  • onigiri-plugin-manager - Plugin management system
  • onigiri-update-notifications - Update notification system
  • onigiri-lockscreen - Lock screen customization
  • onigiri-navbar - Stage Manager with gesture control
